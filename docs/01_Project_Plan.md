# MTB Video Analytics Platform (TrailVision)

## Vision

TrailVision is a desktop application that automatically analyzes mountain bike
GoPro videos using computer vision and artificial intelligence.

The system converts raw GoPro footage into structured ride analytics,
highlight reels, ride statistics, and long-term performance metrics.

---

# Goals

• Automatically import GoPro videos

• Convert videos into an analysis format

• Detect ride events

• Measure riding performance

• Store ride data in DuckDB

• Build analytics models using dbt

• Display dashboards using Power BI
